Platform details & integrations

Everything you need to know about how Aortrix automations work and how to integrate them with your tools.

Features (expanded)

Automated WhatsApp reminders

Use WhatsApp templates to send payment, appointment and renewal reminders. Messages are scheduled and retried based on delivery status.

AI calling agent

Automated calling flows for confirmations and lead qualification. Integrates with call providers (placeholder).

Google review requests

Ask customers for reviews via WhatsApp or via a direct Google review link.

Integrations & architecture

  • WhatsApp Business API - templated messages + session messages
  • Gmail (notifications)
  • Google Sheets (sync & exports)
  • Google Calendar (bookings)
  • n8n (automation flows)

We can provide exported n8n workflows to connect the booking form to Google Sheets, Gmail and Google Calendar. See README for workflow JSON.

Security & hosting

We recommend hosting on Vercel/Netlify for static frontend and using a secure VPS or managed service for backend webhooks and WhatsApp Business API connectivity. Always store credentials in environment variables and use OAuth for Gmail integration.
